# Receipt-Analyzer
An AI-powered web application that extracts grocery purchases from supermarket receipts and provides spending and nutrition insights.

## Motivation
Managing a monthly grocery budget can be surprisingly difficult, especially for students and young professionals. Tracking spending manually is time-consuming, and paper receipts make it hard to analyze purchasing habits over time. This project automates receipt digitization using multimodal AI and presents spending analytics through an intuitive dashboard.

## Features
- Upload supermarket receipts
- AI-powered receipt extraction
- Automatic grocery categorization
- Spending dashboard
- Nutrition overview
- Purchase history
